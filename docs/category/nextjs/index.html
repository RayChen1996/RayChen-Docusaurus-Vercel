📄️ Router Group

在 Next.js 中，Route Group 是用來對 URL 路由進行分組的一種方式，尤其是在 app 目錄中。這種機制允許你在同一層級的路由中將不同的路由集合進行邏輯分組，並且可以不直接影響頁面的結構。這對於多層結構的應用程式有很大的幫助，讓路由設計更加清晰並能夠更方便地進行重構。
