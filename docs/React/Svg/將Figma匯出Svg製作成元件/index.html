如何將設計稿上的SVG圖檔製作成 React 元件？

首先先將滑鼠點到想要匯出的地方，對其按下右鍵，出現選單，選擇Copy as，再點擊次選單，選擇Copy as SVG。 建立一個元件檔案 這邊會建立在 /src/components/Svg/

/src/components/Svg/back.svg
import React from "react";

type SvgProps = JSX.IntrinsicElements["svg"];

function SvgComponent(props: SvgProps) {
  return (

  );
}
export default SvgComponent;
